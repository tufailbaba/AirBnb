## ⚙️ Setup Instructions

1. Clone the repo:

   ```bash
   git clone <your-repo-url>
   cd <repo-name>
   ```

2. Install dependencies:

   ```bash
   bundle install
   ```

3. Setup database:

   ```bash
   rails db:create
   rails db:migrate
   ```

4. Start the server:

   ```bash
   rails server
   ```

5. Visit in browser:

   ```
   http://localhost:3000
   ```
